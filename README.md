# PharmaPays++

Version enrichie du prototype PharmaPays — JavaFX + SQLite + Maven.

## Lancer le projet

Avec Maven :
```
mvn javafx:run
```

Ou via ton IDE en important le projet Maven et en exécutant la classe `gui.MainApp`.

## Notes
- Base SQLite initialisée automatiquement (src/main/resources/db/pharmapays.db).
- Dépendances : sqlite-jdbc, gson, pdfbox.
